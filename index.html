Crystals Collector




Instructions

A number will be generated when the game starts.

Each time you click a crystal you receive a random number of points. To win simply match your points to the number generated! you will lose if your score goes above the random number.

When the game begins again, the player should see a new random number and all the crystals will have four new hidden values.

Each time the game restarts the value of each crystal will change!
